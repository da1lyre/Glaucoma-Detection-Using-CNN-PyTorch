# Glaucoma-Detection-Using-CNN-PyTorch
### Обнаружение глаукомы по съемкам глазного дна
> Бинарная классификация на изображениях с помощью сверточной нейросети (CNN) на PyTorch.

---

## О проекте

Глаукома - одна из главных причин слепоты. Ранняя диагностика по снимкам глазного дна помогает сохранить зрение.

Результаты работы:
- Обучил CNN с нуля для бинарной классификации
- Провёл EDA
- Применил patient-wise split для устранения утечек данных
- Использовал аугментацию и нормализацию

---

## Архитектура модели

Использована **сверточная нейросеть (CNN)**, написанная с нуля на PyTorch.

### Структура

| Слой | Детали |
|------|--------|
| **Conv1** | 3 → 32 канала, ядро 3×3, padding=1, BatchNorm, ReLU, MaxPool 2×2 |
| **Conv2** | 32 → 64 канала, ядро 3×3, padding=1, BatchNorm, ReLU, MaxPool 2×2 |
| **Conv3** | 64 → 128 каналов, ядро 3×3, padding=1, BatchNorm, ReLU, MaxPool 2×2 |
| **Conv4** | 128 → 256 каналов, ядро 3×3, padding=1, BatchNorm, ReLU, MaxPool 2×2 |
| **Flatten** | Выпрямление в вектор |
| **FC1** | Полносвязный слой: 256 × 32 × 32 → 512 (для входного размера 512×512) |
| **Dropout** | 0.5 |
| **FC2** | Выходной слой: 512 → 2 (GON+ / GON-) |

### Трансформации

| Набор | Трансформации |
|-------|---------------|
| **Train** | Resize(512×512), RandomHorizontalFlip, RandomRotation(10°), ColorJitter, ToTensor, Normalize |
| **Val / Test** | Resize(512×512), ToTensor, Normalize |

### Обучение

| Параметр | Значение |
|----------|----------|
| **Функция потерь** | CrossEntropyLoss |
| **Оптимизатор** | Adam, lr=0.001 |
| **Эпохи** | 10 |
| **Batch Size** | 16 |
| **Размер входного изображения** | 512×512 |
